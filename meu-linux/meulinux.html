1 - Após a instalação, dar o primeiro boot e verificar se o sistema carregou e logou!

Configurações do Sistema Pós-instalação:

- Verificar a instalação de pacote de idiomas adicional para PT-BR.
- Configurar Gerenciamento de Energia.
- Verificar uso de pacotes Snap e Flatpak e configurar para não atualizar automaticamente.
- Verificar ou Instalar monitor do sistema e de processos.
- Usar Wallpapers e Icones Próprios Salvos(Opcional).

Utilitários para Sistema ou Configuração:

- Instalar Gerenciador de Pacotes Synaptic.
- Instalar Gparted - Particionador de Disco.
- Instalar impressora em pdf - printer-driver-cups-pdf.
- Instalar Impressora HP Ink Advantage 2676 (ou Epson) e driver, se necessário.
- Instalar Analisador de uso de Disco - Baobab.
- Programa Stacer: Linux System Optimizer and Monitoring.

Programas Internet:

- Instalar Navegador Firefox e configurar sincronização.
- Instalar Navegador chromium (ou chrome), configurar sincronização.
- Instalar pacotes de idiomas do chromium, se necessário - chromium-l10n.
- Instalar Bloqueador de anúncios - chromium-ublock-origin.
- Instalar Aplicativos do Google através de atalhos do chrome-ium (Google Drive, Youtube, Fotos).
- Configurar Google Drive para editar e usar arquivos de modo sem conexão(Offline).
- Instalar Atalhos do Chrome-ium do Netflix - Amazon Prime Video - WhastApp.
- Instalar Cliente torrent qBittorrent ou Deluge.
- Instalar Gerenciador de Downloads uGet.
- Instalar Dropbox para sincronizar pastas remotas do Desktop e Download.

Programas Produção e Trabalho:

- Instalar ePass2003-Castle: Driver para certificado digital Soluti.
- Instalar Java Oracle: Necessário para usar o terminal HOD da Serpro(SIAFI).
- Instalar Terminal HOD Serpro: Acesso ao SIAFI e SIASG.
- Instalar Suite de Aplicativos Office - Libreoffice.

Estudo e Programação:

- Instalar scratch.
- Instalar Visual Studio Code e as Extensões utilizadas
- Instalar Git e GitHub Desktop

Programas adicionais úteis:

- Instalar Media Player VLC: Bom para enviar videos para ChromeCast rapidamente.
- Instalar Inkscape.
- Instalar Fontes ttf-mscorefonts-installer.
- Instalar Virtual Box.
- Instalar Gravador de Imagens em USB.
- Instalar programa de verificador de arquivos (Ver se já tem)- Ex.:GtkHash.
- Instalar CPU-X - Informações sobre o Hardware.

Ações Personalizadas no Gereciador de Arquivos - Thunar ou outro:

Converter pdf para png (empenhos) - Commando: pdftoppm %f -png %f.png
Abrir Terminal como Administrador - Commando: pkexec exo-open --working-directory %d --launch TerminalEmulator (Verificar comando certo)
